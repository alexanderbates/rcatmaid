Plot skeleton and connectors for neuron retrieved from CATMAID

Usage

# S3 method for catmaidneuron
plot3d(x, WithConnectors = FALSE, WithNodes = FALSE,
  ...)

Arguments

x
A neuron to plot
WithConnectors
logical indicating whether or not to plot connectors
WithNodes
logical indicating whether to plot branch/end points (default to FALSE since they will obscure the synapses).
...
additional arguments passed to plot3d.neuron (synapses). Default: TRUE.

Description

Plot skeleton and connectors for neuron retrieved from CATMAID
